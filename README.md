# UTS-Data-Science

Nama : Muhammad Farras Ibnu Budiyanto
Kelas : IF 404
Prodi : Informatika

Dengan belajar data science kita dapat mengambil keputusan bukan lagi secara perasaan, namun secara lebih matang menggunakan data yang telah diolah, data ini dapat menjadi bukti terhadap keputusan yang kita ambil. selain itu belajar data science dapat membantu kita dalam melakukan riset atau penelitian, karena membersihkan data dari data yang kosong atau duplikat secara menual akan terasa lebih melelahkan, dengan adanya data science kita mampu melakukan berbagai pekerjaan yang berhubungan dengan data menjadi lebih efisien.
